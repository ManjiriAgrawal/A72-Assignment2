About StudentProject

This project demonstrates a multi-app Django project with Docker and CI/CD using Jenkins.

🎯 Project Objective

The objective of this project is to create a multi-app Django project that can be containerized with Docker and deployed using a CI/CD pipeline built with Jenkins. It highlights how different components work together in a production-ready environment.

🛠️ Key Technologies Used

  • 🔹 Django: Python framework for building web applications.
  • 🔹 Docker: Containerizes the application for easier deployment.
  • 🔹 Jenkins: Automates the CI/CD pipeline.
  • 🔹 Bootstrap: Adds responsive design and styling.
